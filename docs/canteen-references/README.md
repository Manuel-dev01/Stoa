Pinned excerpts from Canteen's blog posts that shape Stoa's design. These are quoted under fair use for reference and citation purposes. Read the originals at thecanteenapp.com.

## Files in this directory

- `multi-agent-landscape.md` — excerpts from *The Multi-Agent Landscape in 2026* (March 27, 2026), framing the difference between agency, automation, and emergent multi-agent interaction.

When Stoa's documentation or thesis essay cites Canteen, it cites the canonical URL. These local files exist for offline reference during the build, not as a republication.
